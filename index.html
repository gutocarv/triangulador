📖 COMO USAR

  1. Adicionar ponto — informe sua localização (lat/lon), o azimute medido (bússola) e o alvo observado.
  2. Pontos padrão — vizinhos cadastrados aparecem para preenchimento rápido.
  3. Calcular triangulação — o sistema cruza os azimutes e estima a posição do alvo usando centróide ponderado para múltiplas observações.
  4. Gerar mapa — plota pontos, raios de azimute e intersecções no mapa.
  5. Compartilhar — gera um link com todos os dados embutidos para enviar aos colegas.
